Copy and Paste setup:- 

Copy Path Folder -- "ComputerGraphicsLab2024\freeglut" and Paste Folder -- "C:\MinGW"

1. Copy the "include folder" from "ComputerGraphicsLab2024\freeglut" and Paste in "C:\MinGW folder"

2. Copy everything from "ComputerGraphicsLab2024\freeglut\lib" and Paste in "C:\MinGW\lib"

3.Copy "freeglut.dll" from "ComputerGraphicsLab2024\freeglut\bin\x32" and paste inside "your project directory(in our case it's ComputerGraphicsLab2024\CGL in C or ComputerGraphicsLab2024\CGL in C++)".If it's already inside the folder then no need to perform this step.

4.Copy ".vscode folder" from "ComputerGraphicsLab2024\freeglut and paste inside "your project directory(in our case it's ComputerGraphicsLab2024\CGL in C or ComputerGraphicsLab2024\CGL in C++)".If it's already inside the folder then no need to perform this step.
